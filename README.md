## 프롬프트

1. 사용자가 입력한 연도의 히트곡 중 랜덤으로 5곡을 뽑아 게임 진행
2. 선택한 노래에서 하이라이트 부분의 가사 2줄을 반환
3. 실제 존재하는 가사만 반환
